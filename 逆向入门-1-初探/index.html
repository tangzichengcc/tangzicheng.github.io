​ 这个题需要你给一个输入s,然后s被拆成了8个部分,然后这8个部分需要满足一些条件,都满足后,就可以得到flag了

​ 输入的s被拆成了很多部分, 为什么呢…

根据sub_400766可知
v2 = unk_6020B0;
v3 = unk_6020B4;
v4 = unk_6020B8;
v5 = unk_6020BC;

所以s = s + dword_6020A4 + dword_6020A8 + dword_6020AC + unk_6020B0 + unk_6020B4 + unk_6020B8 + unk_6020BC

所以我们的目标就是解出来每个值,然后就得到了正确的输入,输入这个s,就可以得到flag了
